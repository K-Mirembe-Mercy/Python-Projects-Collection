🐍 Snake Game (Python + Pygame)

A simple and fun Snake Game built using Python and Pygame.
This project demonstrates basic game development concepts like movement, collision detection, and score tracking.

---

🎮 Features

- Smooth snake movement using arrow keys
- Random food generation
- Score tracking system
- Game over detection
- Restart option (Press C to play again)
- Quit option (Press Q to exit)

---

🛠️ Technologies Used

- Python 3
- Pygame

---

📂 Project Structure

snake-game/
│── main.py            # Main game file
│── README.md          # Project documentation
│── requirements.txt   # Dependencies

---

▶️ How to Run the Game

1. Install Python (if not installed)

Download from: https://www.python.org/

---

2. Install Pygame

pip install pygame

---

3. Run the Game

python main.py

---

🎯 Controls

Key| Action
⬅️| Move Left
➡️| Move Right
⬆️| Move Up
⬇️| Move Down
C| Restart Game
Q| Quit Game
