🎈 Balloon Shooter Game

A simple and fun balloon shooting game built with Python and Pygame.
Your goal is to pop as many balloons as possible before they escape!

🚀 Features

- 🎯 Aim and shoot balloons
- 🎨 Colorful visuals
- 📈 Live score tracking
- ⚡ Smooth gameplay

---

🕹️ Controls

- Move mouse → Aim
- Click → Shoot

---

▶️ How to Run

1. Install Python
2. Install Pygame:

pip install pygame

3. Run the game:

python main.py
